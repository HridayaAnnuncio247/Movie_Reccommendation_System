Sign_in folder: Has all the logic for the Signup/Login flow, creating the set up page to solve the cold start problem flow and the recommendation flow.

Movie_Database folder: Has the codes for creation of the movie database. This includes, extracting data from TMDB/Wikipedia and writing to the collection.

Models_for_cast_and_crew folder: Has the codes for using word2vec for Cast-crew embeddings, the saved models and the creation of the complete movie embeddings.
